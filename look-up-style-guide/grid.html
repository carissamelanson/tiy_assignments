Grid

To create a simple and responsive layout, I used the grid framework Bourbon Neat. In the smallest media query, the grid for the header and hero is 1 twelve column, with the grid for the content 1 ten column. The medium and large media query have a similar grid framework for the header and hero. The grid for the content on most pages is 10 columns with 1 column to the left and right. On other pages like the connections and the messages pages, the grid changes. On the connections page the grid for the content is 2 five columns. On the messages page the grid is 1 three column and 1 seven column.
